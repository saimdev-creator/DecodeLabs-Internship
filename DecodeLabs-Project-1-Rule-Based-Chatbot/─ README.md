# 🤖 DecodeLabs Rule-Based AI Chatbot
Developer: Muhammad Saim Rizwan (BS Computer Science Student)
A simple Rule-Based AI Chatbot developed in Python as part of **DecodeLabs Industrial Training Kit 2026 (Project 1)**.

The chatbot interacts with users through predefined rules and demonstrates the fundamentals of Artificial Intelligence using control flow, decision-making logic, and dictionaries.

## 📌 Project Overview
This project is designed to simulate a basic AI chatbot using predefined responses instead of Machine Learning or Large Language Models (LLMs).

The chatbot can:
* Respond to greetings
* Answer basic predefined questions
* Handle help requests
* Display dynamic date and time
* Exit the conversation gracefully
* Respond to unknown inputs with a fallback message

## 🚀 Features
- **Rule-Based AI Chatbot Engine:** Deterministic rule resolution mapping.
- **Clean and User-Friendly Interface:** Dark terminal minimalist dark UI designed with native Tkinter components.
- **Continuous Conversation Loop:** Event-driven polling lifecycle loops.
- **Dictionary-Based Knowledge Base:** Memory-efficient text parsing arrays.
- **Input Sanitization:** Structured cleaning handling alphanumeric stripping using standard Regular Expressions (`re`).
- **Greeting Detection & Exit Commands:** Seamless terminal pipeline flow management.
